# IPL Data Analysis

## What this project does
Analysis of IPL cricket matches from 2007-2022 and an ML model that shows diffrence between residual and fitted values

## Key Findings
- Toss winner wins only 51% of matches
- Mumbai Indians are the most successful team
- Winning margins are consistent across seasons
- 461 out of 950 toss losers still won their match

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikitlearn
- Seaborn
